#JavaScipt Data 101
##Writing data to memory in JavaScript can be done with one of three keywords. 

`var` 

```
var name = "Paul";  
seen in older code
```

`let`

```
let name = "Paul"; 
name = "Paula" (reassignment is allowed);
```

`const`

```
const name = "Paul"; 
const name = "Paula" (This is not allowed);
```
